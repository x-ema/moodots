# VR 

Some usefull resources and tweaks that help with vr on linux. 

SteamVR
- Nyabsi's "How to make SteamVR perform well on Linux on AMD systems."
    - https://gist.github.com/Nyabsi/8d055bbda8f4c82100687a77c1440373
- Get SteamVR window to show up on hyprland -> QT_QPA_PLATFORM=xcb %command%
- Disable chaperone for SteamVR -> remove "collision_bounds" from ~/.local/share/Steam/config/chaperone_info.vrchap
- Tweaks to fix some SteamVR bullshit -> https://github.com/BnuuySolutions/SteamVRLinuxFixes

Corectrl 
- make sure the "Virtual Reality" profile is enabled when trying to do vr
